# MCQTSSMusic

### 一个基于Python开发的简易音乐播放器

### 音乐播放API使用QQ音乐,相关API等请参考[QQ音乐解析](https://github.com/MCQTSS/MCQTSS_QQMusic)

### 经测试支持Windows和Debian11 x86_64

### Centos等其他系统并未测试,Linux下UI可能存在部分Bug

## 支持功能:

### 播放音乐,歌词显示&匹配,搜索音乐,加载歌单&专辑列表,QQ音乐登录
